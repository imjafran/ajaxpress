== A brief Markdown Example ==

It shows without reloading:

1. Internal Page, url, post, posts, custom post type, category, tag, custom taxonomy any kind of dynamic permalink
1. Ajax Search, without reloading the page
1. Submit comment to a post / page without reloading the page 

It will ignore:

* External link
* Admin panel dashboard
* Link with new tab
